# Software Support FAQ

## Login and Access

- If a user forgot password: use "Reset password" and check inbox + spam folder.
- If SSO login fails: verify corporate account is active and browser cookies are cleared.

## API Access

- 401 Unauthorized: verify token validity and required scopes.
- 429 Too Many Requests: apply retry with backoff and reduce burst traffic.

## Billing and Plan

- Invoices are available in the billing portal under "Documents".
- Plan changes take effect immediately; prorated differences appear on next invoice.
